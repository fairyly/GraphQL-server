- https://docs.aws.amazon.com/cli/latest/userguide/install-cliv2-docker.html

## install on windows

- 下载 安装 ：https://awscli.amazonaws.com/AWSCLIV2.msi

默认安装在 `C:\Program Files\Amazon\AWSCLIV2`.

- 查看安装是否成功

```
aws --version
```
